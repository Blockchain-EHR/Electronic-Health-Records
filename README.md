\# Blockchain-Based Electronic Health Records



This project is a blockchain-based Electronic Health Records system.



\## Team Responsibilities



\### Member 1 - Backend \& Database

\- Spring Boot API

\- PostgreSQL

\- Authentication

\- File upload

\- Hash generation

\- Blockchain integration



\### Member 2 - Blockchain

\- Solidity smart contracts

\- Hardhat setup

\- Access control

\- Record hash storage

\- Contract tests



\### Member 3 - Frontend

\- React UI

\- Login/Register pages

\- Patient dashboard

\- Doctor dashboard

\- Medical record screens

\- API integration



\## Project Structure



\- frontend/: React application

\- backend/: Spring Boot application

\- blockchain/: Solidity smart contracts

\- docs/: project documentation
